Bootstrap and jQuery Experiment 1

jQuery : Toggle the visibility of Text

I will disappear when you click on Hide!!!

About the Experiment

Sometimes, it is useful to hide some text initially and show it only when a certain button is clicked on a webpage or vice versa. In this experiment, I have tried to explore the toggle visibility feature jQuery using the hide and show methods

I have used the above methods and placed two buttons : "HIDE" and "SHOW". When the Hide button is clicked the text will be hidden. And when the Show button is clicked, the Text is visible again. Try it yourself!!!

Code Snippet

 $(document).ready(function () {

            $("#Hide").click(function () {

                $("h5").hide();

            });

            $("#Show").click(function () {

                $("h5").show();

            });

        });

About the Experiment

We want to display details of different orders. Based on a parameter order_id we will define order details in view.In angular while define route we can define parameters using orderId in url.

And we can read the parameter in ShowOrderController by using $routeParams.orderId. It contains order information data in tabular format. Each order has a “show details” link. This link uses parametrize route url to load order detail screen. Don’t forget to inject $routeParam parameter in controller. Otherwise you wont be able to use it.

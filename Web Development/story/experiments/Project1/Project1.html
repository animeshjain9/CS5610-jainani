About the Experiment

The purpose of this experiment is to get more used to using API's and displaying data properly without using too much CSS and bootstrap. Here I am trying to learn the nitty-gritties of placing elements using CSS. Also, I am learning more about how to get the data given by the API and display it quickly in a format that is very generic and at the same time, it should serve the purpose of giving a good look and feel. I like the tiled pattern of placing my data in the screen and I plan to perfect it for my project.

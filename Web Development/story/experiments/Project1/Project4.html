About The experiment

This experiment is about testing the "@media" property of CSS, accoding to the change in window size, if the size of the window decreases more than 900px, it will change the structure to an image file and then clicking on that will open the menu. Following is the CSS code for the change.
    @media(max-width:901px){
    .nav{
        display:none;
    }
}
@media(min-width:901px){
    .responsiveMenu{
        display:none;
    }
}

Following is the little code from jQuery of showing the menu when we click on the image and fade away the menu if we click somewhere else.
The command e.stopPropagation(); helps to stop fading out of menu since for fade out we are using document. 
     $("#menuBar").click(function (e) {
            $('.onHover').fadeIn(500);
            e.stopPropagation();
        });
        $(document).click(function () {
            $('.onHover').fadeOut(500);
        });
